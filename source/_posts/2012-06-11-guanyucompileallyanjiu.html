前言：这个模块主要用来将源文件编译成字节码文件

假如想整个目录的编译成pyc文件，那么可以这样用：

import compileall
import re



compileall.compile_dir('examples',   #要编译当面目录下的examples目录
    rx=re.compile(r'/\.svn'))  #'rx'自动了一个排除目录， 不编译.svn目录下的文件

注：以上的也可以使用命令行;

python -m compileall -x '/\.svn' examples
