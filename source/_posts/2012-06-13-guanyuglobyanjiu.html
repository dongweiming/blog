前言：glob使用shell规则进行模式匹配

dongwm@linux-dongwm:~> ls test/*   #shell 查找文件 test/file1  test/file2  test/file3

test/testdir: file

import glob
for name in glob.glob('test/*'):  #类似于  ls test/*
    print name

执行结果： dongwm@linux-dongwm:~> python !$ python test.py test/file3 test/testdir test/file2 test/file1

import glob



print 'Named explicitly:'
for name in glob.glob('test/testdir/*'):
    print '\t', name



print 'Named with wildcard:'
for name in glob.glob('test/*/*'):  #通配符*
    print '\t', name

执行结果：

dongwm@linux-dongwm:~> python test.py Named explicitly: test/testdir/file Named with wildcard: test/testdir/file 

import glob
print 'Single Character Wildcard'
for name in glob.glob('test/file?'):  #'?'匹配一个字符
    print name
print 'Character Ranges'
for name in glob.glob('test/*[0-9]'): #匹配一个数字范围0－9
    print name
