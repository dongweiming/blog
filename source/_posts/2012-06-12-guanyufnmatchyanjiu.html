前言：fnmatch是一个底层比较文件名匹配的模块

import fnmatch



pattern = 'FNMATCH_*.PY'  #要匹配的字符串
print 'Pattern :', pattern
print



for name in ['test.py','fnmatch_fn.py','FNMATCH_fn.PY']:
    print 'Filename: %-25s %s' % (name, fnmatch.fnmatch(name, pattern))

执行结果：

dongwm@linux-dongwm:~> python test.py Pattern : FNMATCH_*.PY

Filename: test.py                   False Filename: fnmatch_fn.py             False #大小写敏感，失败 Filename: FNMATCH_fn.PY             True  #匹配成功

注：要强制区分大小写的比较，无论文件系统和操作系统设置，使用fnmatchcase

import fnmatch



pattern = 'FNMATCH_*.PY'
list = ['test.py','fnmatch_fn.py','FNMATCH_fn.PY']
print 'Matches :', fnmatch.filter(list, pattern) #过滤，只留下匹配项


import fnmatch



pattern = 'FNMATCH_*.PY'
print 'Pattern :', pattern
print 'Regex   :', fnmatch.translate(pattern)  # 这事公共的API，用来转换glob模式到一个正则表达式

执行：

dongwm@linux-dongwm:~> python test.py Pattern : FNMATCH_*.PY Regex   : FNMATCH\_.*\.PY\Z(?ms)
