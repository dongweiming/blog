查看和提取rpm，deb文件

Feb 28th, 2011

1 Redhat/Centos/Fedorade的rpm查看和提取 

#rpm -qlp VirtualBox-4.0-4.0.2_69518_openSUSE113-1.i586.rpm
warning: VirtualBox-4.0-4.0.2_69518_openSUSE113-1.i586.rpm: Header V4 DSA/SHA1 Signature, key ID 98ab5139: NOKEY
/etc/init.d/vboxdrv
/etc/init.d/vboxweb-service



/etc/vbox
/lib/modules
/lib/modules/2.6.34.7-0.7-default
...﻿﻿
其中 q -查询RPM文件  l -列出文件  p -指定包的名称

cpio命令是通过重定向的方式将文件进行打包备份，还原恢复的工具，在这里先转换到cpio归档文件，再提取！ 

#  rpm2cpio VirtualBox-4.0-4.0.2_69518_openSUSE113-1.i586.rpm |cpio  -idmv
./etc/init.d/vboxdrv
./etc/init.d/vboxweb-service
./etc/vbox
./lib/modules
./lib/modules/2.6.34.7-0.7-default
...

# dpkg -x lcrt_0.0.8_i386.deb lcrt  //提取文件到lcrt目录下 linux-3qd1:/home/dongwm/下载 # cd lcrt/ DEB是ar文件，他总是包含三个文件-- debian-binary，control.tar.gz ，data.tar.gz，可以使用ar 提取data.tar.gz  再解压data.tar.gz 
# ar -vx lcrt_0.0.8_i386.deb
